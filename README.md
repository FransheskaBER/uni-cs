## Courses

### Intro to Computer Science (`intro_CS/`)

Foundational CS theory — how computers represent and process information.

| Topic | Content |
|---|---|
| Information Representation | Data types, number systems, non-numerical data, redundancy & error correction |
| Data Processing | Notes on how data is processed at a system level |

---

### Data Structures (`data_structures/`)

Core Java data structures and programming fundamentals.

| Unit | Topic |
|---|---|
| 1 | Programming style — code conventions, annotations, Javadoc, comments |
| 2 | Objects — `equals`, `toString`, `clone`, `compareTo`, comparison operators |
| 3 | External packages & the Java class library |
| 4 | Collections — Arrays, Lists, Sets, Maps, Queues, Stacks |
| 5 | Strings & Calendar — string manipulation, `StringBuffer`, date/time API |
| 6 | File system & data streams — reading/writing files |

Each unit has a `exercises/` folder with hands-on Java practice problems.

---

### Object-Oriented Programming (`java_oop/`)

Deep dive into OOP design and Java language features.

| Unit | Topic |
|---|---|
| 1 | Intro to OO system development |
| 2 | OO modeling |
| 3 | Designing and programming classes |
| 4 | Java language constructs |
| 5 | Inheritance — `Employee`, `Manager`, `Developer`, `Vehicle` hierarchies |
| 6 | Key concepts — polymorphism, abstraction, interfaces |
| 7 | Constructors — `Person`, `BankAccount`, `Student` |
| 8 | Exception handling — custom exceptions, `OnlineShop`, `OrderProcessor` |
| 9 | Interfaces |
